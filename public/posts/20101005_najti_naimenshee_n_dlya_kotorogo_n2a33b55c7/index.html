1 minute read

Все натурально :)

если представить

c = ( 2 ^x1 * 3^y1 * 5^z1 )

b = ( 2 ^x2 * 3^y2 * 5^z2 )

a = ( 2 ^x3 * 3^y3 * 5^z3 )

то из

3b^5=5c^7 => 7x1 = 5x2

а из

5c^7=2a^3 => 7x1 = 3x3 + 1

и по аналогии

7y1 = 5y2 + 1

7y1 = 3y3

7z1 + 1 = 5z2

7z1 + 1 = 3z3

Ищем минимальные x1 x2 x3 y1 y2 y3 z1 z2 z3:

x1 = 10

x2 = 14

x3 = 23

y1=3

y2=4

y3=7

z1=2

z2=3

z3=5

n=52^(107)3^(37)5^(27)=2^703^215^15
