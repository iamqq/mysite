Все натурально :)



если представить

c = ( 2 ^x1 * 3^y1 * 5^z1 )

b = ( 2 ^x2 * 3^y2 * 5^z2 )

a = ( 2 ^x3 * 3^y3 * 5^z3 )



то из

3*b^5=5*c^7 => 7*x1 = 5*x2

а из

5*c^7=2*a^3 => 7*x1 = 3*x3 + 1



и по аналогии

7*y1 = 5*y2 + 1

7*y1 = 3*y3

7*z1 + 1 = 5*z2

7*z1 + 1 = 3*z3

Ищем минимальные x1 x2 x3 y1 y2 y3 z1 z2 z3:

x1 = 10

x2 = 14

x3 = 23



y1=3

y2=4

y3=7



z1=2

z2=3

z3=5






n=5*2^(10*7)*3^(3*7)*5^(2*7)=2^70*3^21*5^15
